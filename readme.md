Project Description - LeakSense is a smart water pipeline monitoring system that detects leakage by comparing the flow of water at the inlet and outlet. It uses ultrasonic sensors to track water levels in containers and a turbidity sensor to analyze water quality. The system helps identify water loss and potential contamination in real time.
